\section{Describing dependable systems }

\subsection{Scenario}

Consider the scenario of a `smart' flower pot which monitors light levels, soil
moisture, soil acidity, as well as automatically waters the soil if needed.

This smart flower pot (\textbf{system C}) consist of, among other things, a
sensor array (\textbf{system B}). This sensor array contains sensor for
acidity, light, and moisture (\textbf{system A}).
